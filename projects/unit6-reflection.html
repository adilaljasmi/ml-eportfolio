üí° Critical Analysis & Learning Outcomes

What Went Well:

  • Audience Adaptation: Successfully translated technical metrics (RMSLE, R¬≤) into business language (prediction accuracy, revenue opportunity)
  • Visual Hierarchy: Used color, size, and positioning to guide executive attention to key insights
  • Actionable Insights: Quantified specific revenue opportunities ($35/night for Manhattan premium properties)
  • Clean Code: Developed reusable functions for feature name cleaning and data transformation

Challenges Overcome:

Challenge 1: Feature importance values didn't sum to 100%, making them confusing for executives

Solution: Normalized the top 10 features to create percentage-based importance metrics


Challenge 2: Technical feature names from the ML model were not interpretable

Solution: Created a comprehensive cleaning function to translate all feature names


Challenge 3: Static visualizations didn't engage the audience effectively

Solution: Switched to Plotly for interactive, explorable visualizations

Areas for Improvement:

  • Could have included time-series analysis for seasonal pricing patterns
  • Dashboard could benefit from real-time data integration
  • Additional user personas beyond executives (hosts, data scientists)
