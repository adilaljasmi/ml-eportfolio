Unit 11: Deep Learning for CIFAR-10 Object Recognition

Project Overview

Type: Individual Project

Objective: Design and implement CNN architectures for image classification on CIFAR-10 dataset

Technologies: Python, TensorFlow/Keras, Google Colab, NumPy, Matplotlib, Plotly

Key Achievements

  • Achieved 88.34% test accuracy with custom CNN
  • Outperformed transfer learning (EfficientNetB0) by 31%
  • Tested 9 hyperparameter configurations
  • Analyzed 4 different CNN architectures
  • Discovered augmentation harmful for 32x32 images
  • Achieved 99.58% top-5 accuracy
  • Training time: 14 minutes on T4 GPU
  • Model size: 1.47M parameters (optimal)

Technical Deep Dive

Models Compared:

  • Custom CNN: 88.34% accuracy - Best performer
  • Transfer Learning (EfficientNetB0): 56.96% accuracy - Failed due to resolution mismatch
  • Architecture Variations: Deep, Wide, Shallow, Residual-like

Key Findings:

  • Deep architecture (1.47M params) optimal vs Wide (8.84M params)
  • Data augmentation decreased performance by up to 52%
  • Best configuration: Adam optimizer, LR=0.001, Dropout=0.5
  • Model excels at vehicles (94%+ accuracy) but struggles with cats/dogs (72-75%)

Critical Insights

This project revealed crucial lessons about deep learning:

  • Resolution Matters: 32Ã—32 images too small for standard augmentation techniques
  • Architecture > Parameters: More parameters doesn't mean better performance
  • Transfer Learning Limitations: Pre-trained models fail when domain mismatch is severe
  • Confidence Calibration: Custom CNN showed 99% confidence when correct vs 72% when incorrect
